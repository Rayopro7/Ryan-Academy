Lesson 3: Structuring Content with Headings and Paragraphs

Now that we have the basic HTML document structure (the skeleton!), let's learn how to add the most fundamental types of visible content: headings and paragraphs. These tags are crucial for organizing your text, making it readable for users, and providing semantic meaning for browsers and search engines.

Paragraphs (`

The `

  • Purpose: It groups related sentences together, forming a distinct block of text.
  • Block-Level Element: Paragraphs are "block-level" elements. This means they typically start on a new line and take up the full available width of their container.
  • Spacing: Browsers automatically add some vertical space (margin) before and after each paragraph, separating it visually from surrounding content.
  • Usage: Simply wrap your paragraph text within `

    ` and `

    ` tags.

Example:

<p>This is the first paragraph. It contains several sentences discussing a particular idea. Browsers will render this as a distinct block.</p>

<p>This is the second paragraph. Notice how it starts on a new line and has space separating it from the first paragraph. This helps readability.</p>

Using headings and paragraphs correctly is fundamental to creating well-structured, readable, and accessible web pages. Always think about the logical structure of your content when choosing which heading level to use.
